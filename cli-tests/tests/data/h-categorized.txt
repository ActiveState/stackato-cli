Getting Started
    stackato login [<name>]       Log in to the current or specified target
                                  with the named user
    stackato logout [<target>]    Log out of the current, specified, or all
                                  targets
    stackato target [<url>]       Set the target API endpoint for the client,
                                  or report the current target

Applications
    stackato app [<application>]    Show the information of the specified
                                    application
    stackato apps                   List the applications deployed to the
                                    target
    stackato list                   List the applications deployed to the
                                    target

    Information
