Getting Started
    stackato login [<name>]       Log in to the current or specified target
                                  with the named user
    stackato logout [<target>]    Log out of the current, specified, or all
                                  targets
    stackato target [<url>]       Set the target API endpoint for the client,
                                  or report the current target

Global Options
    --cafile CAFILE             Path to a file containing the CA certificates
                                to use for validation.
    --color MODE                Set color mode (always|auto|never) of the
                                client. Default is auto, which activates color
                                when in a proper terminal.
    --help                      Show the help of the client (main or current
                                command), and stop.
    --no-prompt                 Disable interactive queries.
    --no-skip-ssl-validation    Complementary alias of --skip-ssl-validation.
    --non-interactive           Alias of --no-prompt.
    --noprompt                  Alias of --no-prompt.
    --skip-ssl-validation       Disable SSL certificate validation warnings.
    --version                   Print the version number of the client, and
                                stop.
    -?                          Alias of --help.
    -h                          Alias of --help.
    -n                          Alias of --no-prompt.
    -V                          Alias of --version.
    -v                          Alias of --version.
